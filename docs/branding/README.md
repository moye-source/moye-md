# 墨页 Moye Branding

## Name

- 中文名：墨页
- English name: Moye
- Positioning: a native macOS Markdown editor for focused local writing.

## Logo Concept

The first logo concept is saved at:

```text
docs/branding/moye-logo-concept.png
```

The mark uses a dark graphite app-icon base, an off-white document form, a pen-nib cue, and a blue Markdown `#` accent. It is intended as an app-icon direction rather than a final vector asset.

## Palette

- Graphite: `#1C1C1E`
- Paper: `#F5F2EA`
- Accent Blue: `#0A84FF`
- Ink: `#111111`

## Usage Notes

- Keep the icon readable at 16px before adding detail.
- Use `墨页` as the primary name in the Chinese interface.
- Use `Moye` in English UI, bundle display copy, and export metadata.
